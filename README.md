# Employee Management System

A Full Stack Employee Management System developed using:

- React.js
- Spring Boot
- MySQL
- Docker
- Docker Compose

---

# Technologies Used

## Frontend
- React.js
- Axios
- Bootstrap
- JavaScript
- JSX

## Backend
- Spring Boot
- Spring Data JPA
- Hibernate
- REST API
- Maven

## Database
- MySQL

## DevOps / Containerization
- Docker
- Docker Compose

---

# Features

## Employee Operations
- Add Employee
- Get All Employees
- Update Employee
- Delete Employee

## Frontend Features
- Employee Table UI
- API Integration using Axios
- React Hooks (`useState`, `useEffect`)
- Responsive Bootstrap Design

## Backend Features
- REST APIs
- Layered Architecture
- DTO & Mapper Pattern
- JPA Repository
